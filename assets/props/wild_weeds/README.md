# Yabani Otlar

Bu paket yalnızca dekoratif dünya varlıklarını içerir. Varlıklar çarpışmasız, etkileşimsiz ve deterministik chunk yerleşimiyle kullanılır.

- Grup: `yabani_otlar`
- Çeşit: 10
- Atlas: 800×320 PNG, gerçek alfa şeffaflık
- Hücre: 160×160, 5 sütun × 2 satır
- Fizik: kapalı
- Oynanış etkisi: yok
- Yerleşim: chunk üretiminde alan rezervasyonu; birbirinin içine doğmaz
- Performans: tek temel atlas dokusu, mevcut streamed prop pool ve texture cache
